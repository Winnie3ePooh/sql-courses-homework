1. Получение информации о всех атрибутах(учитывая только атрибутную группу и атрибутные типы)
2. Получение атрибутов для конкретного объектного типа(вывести также имя объектного типа и имена атрибутов)
3. Получение иерархии ОТ(объектных типов)  для заданного объектного типа(нужно получить иерархию наследования)
4. Получение вложенности объектов для заданного объекта(нужно получить иерархию вложенности)
5. Получение объектов заданного объектного типа(учитывая только наследование ОТ)
6. Получение значений атрибутов для заданного объекта(без учета наследования ОТ). Вывести в виде(ID_атрибута – Имя_атрибута – Значение_атрибута)
7. Получение ссылок на заданный объект(все объекты, которые ссылаются на текущий)
8. Получение значений атрибутов(без повторяющихся атрибутов) для заданного объекта( с учетом наследования ОТ) Вывести в виде см. п.6
